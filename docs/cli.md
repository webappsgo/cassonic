# CLI Reference

`cassonic-cli` is the companion CLI tool for the cassonic server.

## Usage

```bash
cassonic-cli [flags] [command]
```

## Flags

| Flag | Description |
|------|-------------|
| `--help`, `-h` | Show help |
| `--version`, `-v` | Show version |
| `--server {url}` | Server URL |
| `--debug` | Enable debug output |
| `--color {auto\|yes\|no}` | Color output control |
