Help

Getting Started

cassonic listens on port {{.Port}} by default. Replace yourserver with your hostname or IP.

Connect a Subsonic Client

Use these settings in any Subsonic-compatible app (DSub, Ultrasonic, Symfonium, Clementine, Rhythmbox, Amarok, etc.):

  • Server URL: http://yourserver:{{.Port}}/rest
  • Username / Password: your cassonic credentials
  • API version: any (1.1.0 through 1.16.1 supported)

Connect an Ampache Client

Configure your Ampache-compatible client with:

  • Server URL: http://yourserver:{{.Port}}/server/json.server.php
  • API version: 5 or 6 (XML and JSON both supported)

Native REST API

The native API is versioned at /api/v1/ and returns JSON.

GET http://yourserver:{{.Port}}/api/v1/health
→ {"ok":true,"data":{"status":"ok"}}

CLI Usage

cassonic-cli --server http://yourserver:{{.Port}} login
cassonic-cli --server http://yourserver:{{.Port}} scan
cassonic-cli --server http://yourserver:{{.Port}} status

Icecast Streaming

After adding an Icecast server in Icecast settings, streams are available at:

http://icecast-host:8000/<mount>

Mounts relay by all tracks, artist, or genre — configurable per mount.
